# ♟️ Jogo de Damas em HTML, CSS e JavaScript

Um jogo de **damas** totalmente funcional, desenvolvido em **HTML, CSS e JavaScript puro**, dentro de **um único arquivo HTML**.  
Inclui todas as regras clássicas: movimentos diagonais, capturas obrigatórias, múltiplas capturas e promoção para dama (rei).

---

## ✨ Funcionalidades
- 🎨 Tabuleiro estilizado com cores e gradiente de fundo
- 🔴 Peças vermelhas e ⚫ pretas bem desenhadas
- ➡️ Movimentos diagonais válidos
- ⚔️ Capturas obrigatórias e múltiplas
- 🥇 Promoção para dama com destaque dourado
- 🏆 Verificação automática de vitória

🛠️ Tecnologias utilizadas

    HTML → Estrutura do tabuleiro

    CSS → Estilo visual e design das peças

    JavaScript → Lógica de movimentos e regras
